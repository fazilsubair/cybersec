What is VulnScanner?

VulnScanner is an open-source security testing tool that allows you to perform automated vulnerability checks on various services using customizable YAML templates. It is designed to be easy to use while also offering flexibility for more advanced use cases.

Core Features:

  • Template-Based Testing: Define checks for HTTP status codes, response bodies, headers, and more using simple YAML configuration files.
  • Customizable: Create and upload your own templates, or use the pre-built ones to start testing immediately.
  • Supports Multiple Services: VulnScanner can test services over HTTP, FTP, SMTP, and other protocols.
  • Open Source: VulnScanner is completely open-source, allowing you to contribute or modify it according to your needs.
