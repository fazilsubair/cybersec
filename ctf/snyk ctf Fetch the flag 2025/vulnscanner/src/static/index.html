What is VulnScanner?

VulnScanner is a versatile, open-source vulnerability scanner that allows you to create and upload custom templates to perform checks on various services. It uses YAML-based templates to define security checks such as HTTP status codes, response bodies, and other network parameters.

The scanner performs these checks by sending requests to the services you specify and comparing the responses with your predefined expectations. It's an essential tool for automating vulnerability testing in your development or production environments.

Key Features:

  • Create custom templates to define checks
  • Support for various services (HTTP, FTP, SMTP, etc.)
  • Easy-to-use interface for uploading and managing templates
  • Flexible YAML configuration for advanced use cases

Get Started

Begin by exploring the available templates or uploading your own to start testing your services. Whether you're looking for HTTP checks or testing other services, VulnScanner makes it easy to create, customize, and manage your security tests.
